# Tic Tac Toe POC

A browser based tic tac toe proof of concept built with HTML, CSS, and JavaScript.

## Live Demo

https://white-shepherd.github.io/tic-tac-toe-poc/

## What it does

- Renders a playable tic tac toe board
- Tracks player turns
- Detects wins
- Detects draws
- Allows the game to reset
- Tracks X wins, O wins, and draws during the browser session
